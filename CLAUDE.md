# CLAUDE.md

> `CLAUDE.md` and `AGENTS.md` are autogenerated by `/init`. Do not edit directly.

Microsoft 365 integration — Outlook email, Teams, SharePoint, OneDrive, Planner via `m365` CLI.

## Skills

Skills in `skills/*/SKILL.md`: EdgeBrowser, M365. M365 provides Outlook email operations (list, read, move, compose, attachments). See `CONVENTIONS.md` for m365 CLI patterns, auth flow, and gotchas.

## Install / Verify

```bash
make install          # deploy skills (Claude, Gemini, Codex, OpenCode)
make verify           # check skills deployed across all providers
make clean            # remove previously installed skills
make test             # validate-module convention checks
```

### Tool

[m365 CLI](https://pnp.github.io/cli-microsoft365/) — install via `npm install -g @pnp/cli-microsoft365`.
